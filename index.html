Học GIT

Có 2 cách để tải mã nguồn từ github:

- git clone + đường dẫn file

- download toàn bộ dạng zip file

Cách đưa code lên GitHub:

* Tạo một repository mới trên Github (trên web)

- Sau khi các bạn đã thực hiện bước 1 xong, các bạn đăng nhập vào GitHub và tạo một repository mới để đưa code lên GitHub. Các bạn thực hiện giống như ảnh bên dưới:

- Chọn New repository. Sau đó các bạn nhập thông tin bên dưới như hình:

- Sau khi các bạn chọn Create repository thì sẽ hiện được như hình bên dưới đây:

Đưa code lên GitHub (dưới desktop)

- Đầu tiên các bạn mở VSCode lên vào thư mục mà bạn muốn đưa code lên GitHub. Các bạn vào Terminal chọn New Terminal:

- git init (1 git repository 1 project mới)

Sau đó ta gõ git status để check những file chưa được đưa lên hoặc đã thay đổi trong thư mục(không bắt buộc)

- git add ./ (tất cả những file cần đưa lên github)

Để kiểm tra mình đã add thành công hay chưa các bạn gõ git status lại để check(không bắt buộc)

- git commit -m "commit 1"(Lệnh này dùng để note lại các thay đổi để dễ dàng theo dõi)

- git branch -M main

- git remote add origin https://github.com/huynhvanhoa/git.git

- git push -u origin main (đưa code lên Github)
